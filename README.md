# 🎮 KBC Game (Python)

A simple command-line based **Kaun Banega Crorepati (KBC)** quiz game built using Python.
Test your knowledge, answer questions, and win virtual prize money! 💰

---

## 🚀 Features

* Multiple-choice quiz questions
* Increasing prize levels
* Lifelines (if implemented)
* Input validation
* Score tracking
* Simple and interactive CLI interface

---

## 🛠️ Technologies Used

* Python 3

---

## 📂 Project Structure

```
kbc-game/
│── kbc.py          # Main game file

```

---

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/kbc-game.git
```

2. Navigate to the folder:

```
cd kbc-game
```

3. Run the game:

```
python kbc.py
```

---

## 🎯 How to Play

* You will be asked multiple-choice questions.
* Each correct answer increases your prize money.
* Choose the correct option (A/B/C/D).
* Game ends when you answer wrong or complete all questions.

---

## 🧠 Example Question

```
Q1: What is the capital of India?
A. Mumbai
B. Delhi
C. Chennai
```
